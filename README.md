# RandomTaskGenerator
## Описание программы
"RandomTaskGenerator" – это удобное приложение с графическим интерфейсом (GUI), предназначенное для генерации случайных задач. Приложение позволяет вести учет сгенерированных задач, фильтровать их по типу (например, "Учёба", "Спорт", "Работа") и сохранять историю в файле JSON. Это поможет вам не забывать о делах и поддерживать продуктивность.

## Инструкция по установке и запуску

### Требования:

*   Python 3.6 или выше.
*   Библиотеки: `tkinter`, `random`, `json`, `os`. Все эти библиотеки входят в стандартную поставку Python, поэтому установка дополнительных пакетов не требуется.
*   Git (для контроля версий, если вы решили его использовать).

### Инициализация Git-репозитория (опционально):

Если вы планируете использовать Git для контроля версий, перейдите в директорию проекта в терминале и выполните:
bashgitinit
### Запуск приложения:

1.  **Сохраните исходный код:** Скопируйте предоставленный код приложения и сохраните его в файле с расширением `.py` (например, `random_task_generator.py`).
2.  **Откройте терминал или командную строку:**
3.  **Перейдите в директорию:** Используйте команду `cd`, чтобы перейти в папку, где вы сохранили файл с исходным кодом.
4.  **Запустите приложение:** Выполните команду:
    ```bash
python random_task_generator.py
    ```
5.  **Сохранение данных:** История сгенерированных задач будет автоматически сохраняться в файле `task_history.json`. Стандартный набор задач (с типами) будет сохранен в `tasks_data.json` при первом запуске, если эти файлы не существуют.

## Примеры использования

*   **Генерация задачи:** Нажмите кнопку "Сгенерировать задачу", чтобы получить новую случайную задачу. Задача будет отображена во всплывающем окне и добавлена в историю.
*   **Фильтрация:** В выпадающем списке "Фильтр по типу" выберите нужную категорию задач (например, "Учёба"). Будут отображаться только задачи этого типа. Выберите "Все", чтобы увидеть полный список доступных задач.
*   **История задач:** Список всех сгенерированных задач отображается в нижней части окна.
